# SoleStyle - E-commerce Shoe Store

## Overview

SoleStyle is a simple e-commerce shoe store built using React and Vite. Users can browse shoes, search and filter products, add items to the cart or wishlist, and complete the checkout process. The project is fully responsive and uses Context API for state management.

## Features

* View all shoes in a responsive product grid
* Product details with images, sizes, quantity selection, and description
* Add and remove items from the shopping cart
* Wishlist to save favorite products
* Checkout form with order summary
* AI chatbot (ShoeBot) for common customer questions
* Search products by name
* Filter products by category, brand, size, and price
* Sort products by price, rating, and newest
* Login and signup modal (mock authentication)
* Guest checkout option
* User data, cart, and wishlist are saved using LocalStorage
* Responsive design for mobile, tablet, and desktop

## Technologies Used

* React 18
* Vite
* React Router DOM
* Context API
* JavaScript (ES6+)
* HTML5
* CSS3
* Font Awesome
* Google Fonts (Inter)
* LocalStorage
